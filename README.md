# Guitar Practicer

This app selects a random practice drill for you, based on some selections.


## Variables

* Chord root
* Chord quality (Major, Minor, Augmented, Diminished)
* Chord augment (none, 2, 4, 7)
* Position (0-14)
* Voice (1-3)
* Technique
  * Strum
  * Pluck
  * Arpeggiate
  * Tap
* Tempo

## Output

Plays the chord root at the tempo.
